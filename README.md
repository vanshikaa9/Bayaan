# Bayaan

Bayaan is an Agentic vernacular product discovery assistant built for Bharat.

Features:
- Natural language product search
- Voice input
- Hindi and English support
- Typo correction
- Query replanning when exact matches are unavailable
- JSON-based product catalog

Tech Stack:
- Python
- Ollama (Qwen 2.5)
- SpeechRecognition
- SoundDevice

Upcoming:
- Flask UI
- Regional language support
- Conversational follow-up questions
